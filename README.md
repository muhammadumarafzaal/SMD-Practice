# SMD Practice

Welcome to my Mobile Application Development (SMD) practice repository! This repository contains various Android and mobile development exercises, projects, and UI practice code.

## 📱 About This Repository
This repository serves as a personal playground for exploring Android development, UI/UX design concepts, database integration, and more. Throughout these projects, I demonstrate an evolving understanding of the Android ecosystem.

### Highlights
- **UI Design & Development**: Exploring complex layouts and custom UI elements.
- **Firebase Integration**: Practical implementations of Firebase Authentication, Realtime Database, and Cloud Messaging.
- **Fragments & Activities**: Managing lifecycle, data passing, and navigation.
- **Recycler Views**: Implementing efficient list rendering.
- **Local & Remote Databases**: Experience with Room, SQLite, and cloud data syncing.
